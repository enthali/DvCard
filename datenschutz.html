Mobile Apps

DvCard App

Die DvCard App speichert alle Daten ausschließlich lokal auf Ihrem Gerät:

  • Visitenkartendaten: Alle von Ihnen eingegebenen Kontaktinformationen (Name, Position, Firma, Telefon, E-Mail, Website, Adresse) werden nur in der lokalen App-Datenbank auf Ihrem Gerät gespeichert.
  • QR-Code-Generierung: QR-Codes werden lokal auf Ihrem Gerät im vCard-Format erstellt. Es erfolgt keine Übertragung dieser Daten an externe Server.
  • App-Einstellungen: Benutzereinstellungen werden nur lokal in den App-Präferenzen gespeichert.

Wichtig: Es erfolgt keine Übermittlung personenbezogener Daten an unsere Server oder Dritte. Alle Daten verbleiben auf Ihrem Gerät.

Datenschutz bei der QR-Code-Nutzung

Wenn Sie einen von der App generierten QR-Code teilen oder von anderen scannen lassen:

  • Der QR-Code enthält nur die von Ihnen freigegebenen Kontaktdaten im standardisierten vCard-Format
  • Sie entscheiden selbst, welche Informationen Sie in der App eingeben und somit im QR-Code verfügbar machen
  • Das Scannen des QR-Codes durch Dritte liegt außerhalb unseres Einflussbereichs
